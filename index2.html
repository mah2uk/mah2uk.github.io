Personalised Kraft Gift Tags with Optional Eyelet and Twine

Item Description

This listing is for a set of Kraft brown gift tags with your own text printed on them in the font provided as per the listing images. These are ideal for all occasions and make any gift or item that little bit more special with your own personalised text. You can choose the pack size you require along with optional Silver eyelet or twine

Please read below for detals on sample tags and how to provide the text you want printed. Please read this carefully to ensure we can provide your tags with no unecessary delay.

Availble Pack sizes are - 1 Sample tag (see details below on sample tags), Single Personalised Tag, Set of 9, 15, 18, 24 or 27. For larger orders where Twine is selected the Twine will be included but will not be looped through the tags.

Specifications:

  • Colour: Kraft Brown (only 1 sided. The back is a generic grey card)
  • Length: 71mm
  • Width: 51mm
  • Thickness: 280gsm card
  • Optional Extras: 4mm Silver Eyelet and/or Length of Twine (approx. 260mm/26cm, *13cm when folded*)
