Personalised A6 Wedding Invites

For matching Wedding 'Save the Date' cards please see my other items using the button below:

This listing is for a set of A6 Wedding Invites with your personalised text printed on them in the font as per the listing images. You can choose if you would like these supplied with matching size C6 craft brown envelopes or not. Ideal for those wanting to have more control over their Wedding invites.

Please read below for details on sample invites and how to provide the text you want printed. Please read this carefully to ensure we can provide your invites with no unnecessary delay.

Available Pack sizes are - 1 Sample Invite (see details below on samples), Single Personalised Invite, Set of 8, 16 or 24. If you require more please adjust the quantity as required.

Specifications:

  • Colour: White Laid (Textured) Card
  • Height: 148mm
  • Width: 105mm
  • Thickness: 280gsm card

SAMPLE INVITES

Samples will be provided with only sample text as per the listing photos, to allow you to judge the style and sizes.

Please do not ask for custom text to be printed on a sample, as this will not happen and you will receive the sample with the text as listed in the images.

If you would like a single invite with personalised text printed on it please choose the option for 'Single Personalised Invite' and follow the personalisation instructions below.

I can produce a mockup for any larger orders. Please contact me prior to placing the order or alternatively state this in the message to seller so I can ensure this is provided before producing your invites.

PERSONALISATION INSTRUCTIONS

(PLEASE READ CAREFULLY)

Please supply the text you require printed on your invite via the message/note to seller when placing your order. This will allow me to match your order with the text you require. Failing to do this may result in a delay to your order. Please supply the following details:

For the front -

  • The 2 names you require in the large font.

  • The Date & Time required.

  • The Venue name and/or address.

For the back -

  • The travel / accommodation details (if required).

  • The Events/Activities and how you want them listed (i.e. with times etc.. if required).

  • The RSVP details such as Name, phone number and/or email address for RSVP.

If you do not require any section to be included please state this (i.e. 'Travel/Accommodation' section not required) and I will delete this section. If you do not provide the details for a particular section of the invite I will assume it not needed so please ensure you provide all required info to ensure the invite arrives accurately. Thank you.
